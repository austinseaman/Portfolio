DJ Bentley Taylor
A fully responsive, website for a Salt Lake City, Utah based DJ, for promotional and event booking purposes. End users are able to check out mixes found on the client's Mixcloud page, through an iframe player, and are able to book him for events, through an EmailJS integration.

Tech Toolbox: React, Axios, Mongoose, MongoDB, EmailJS

Kaya Poore
(REBUILDING)
A fully responsive, web app and online store that I built for a local painter, using MERN stack technologies. A DB contains products and, upon admin login, they are able to list additional pieces and update existing DB. Integrated with EmailJS and Stripe API to take inquiries via email and payments for products.

Tech Toolbox: React, Axios, Mongoose, MongoDB, EmailJS, Stripe API

V Trade
A collaborative project with 4 other developers, that allows for college students to sell their outdoor recreational gear to make some extra money to make it through the semester, without going hungry. Deployed through Heroku.

Tech Toolbox: React, Axios, Mongoose, MongoDB, Adobe Spark

TrackOcean
(In the process of being redesigned, to be used with Spotify API.)
A genre discovery app. The user can choose a genre in which they have interest, set the limit for how many tracks they'd like to return, and the app makes a call to the Openwhyd.org music API, to get the site's most popular tracks within that genre, according to user data.

Tech Toolbox: React, Axios, Openwhyd.org API
