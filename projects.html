A fully responsive, scalable web app and online store that I built for a local painter, using MERN stack technologies. A mongoDB database contains the artists products and, upon admin login, they are able to list additional pieces and update/delete existing products. Tech Toolbox: React, Axios, Mongoose, MongoDB, EmailJS, Stripe API
